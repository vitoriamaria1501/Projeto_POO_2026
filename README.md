# Projeto_POO_2026
Este repositório contém a documentação do projeto desenvolvido para a disciplina de POO (programação orientada a objetos)

# Projeto 
Sistema de Agenda de Contatos com Login de Usuário.

O sistema permitirá que usuários criem uma conta, façam login e gerenciem uma agenda de contatos, podendo adicionar, editar, visualizar e excluir contatos.

# Funcionalidades
- Cadastro de usuário
- Login no sistema
- Cadastro de contatos
- Edição de contatos
- Exclusão de contatos
- Listagem de contatos
