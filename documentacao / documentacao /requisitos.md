# Requisitos Funcionais

1 – O sistema deve permitir o cadastro de usuários.
2 – O sistema deve permitir que o usuário realize login.
3 – O sistema deve permitir adicionar novos contatos.
4 – O sistema deve permitir editar contatos existentes.
5 – O sistema deve permitir excluir contatos.
6 – O sistema deve permitir visualizar a lista de contatos cadastrados.
7 – O sistema deve armazenar os contatos no banco de dados

# Requisitos Não Funcionais

1 – O sistema deve ser desenvolvido em Java.
2 – A interface gráfica deve utilizar Java Swing.
3 – O banco de dados deve ser MySQL.
4 – O sistema deve possuir interface simples e intuitiva.
5 – Os dados devem ser armazenados de forma persistente no banco de dados.
